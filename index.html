Arka BETA – wersja studia

Wewnętrzna wersja z pełnym pipeline’em treningu neuronowego. Nie jest częścią produktu dla klienta – służy nam do rozwijania wiedzy zwierzaków i modeli.

Pipeline treningu neuronowego

Kolejność: przemiał → psucie → pary zadań → neuronowa run 1 → neuronowa run 2.

  1. Przemiał całość – Ollama 32B poprawia lekcje (oryginał→poprawiony).
  2. Psucie całość – Ollama 14B „psuje” lekcje; pary: zepsuta→oryginał.
  3. Pary zadań – Run 1 i Run 2 (więcej wariantów + pytania łączone).
  4. Neuronowa 1. run – eksport phase 1 → full fine-tuning 4B.
  5. Neuronowa 2. run – eksport phase 2 → fine-tuning 4B.

Dzień Treningowy, eksport danych, skrypty CLI. Na 32 GB RAM trening 4B wchodzi; 7B – opcja LoRA w dokumentacji.
